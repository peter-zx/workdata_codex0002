# Daily Log Tool

一个电脑和手机自适应的多人工作日志小工具。伙伴注册登录后，按条记录「工作内容、工作进度、复盘优化」，系统自动拼合为今日工作日志，并支持一键复制和年月周日视图回看。

## 启动

```powershell
npm start
```

打开：

```text
http://localhost:3000
```

注册登录和数据隔离依赖服务端接口，请通过 `http://localhost:3000` 访问，不要再用 `file://` 直接打开。

## 功能

- 单条记录由三个字段构成：工作内容、工作进度、复盘优化。
- 三个字段都不必填，填写任意字段即可保存。
- 支持注册、登录、退出。
- 每个账号只能看到自己的工作记录。
- 自动按日期归档，支持日、周、月、年视图。
- 生成「今日日志」文本，一键复制。
- 数据保存在服务端 `data/` 目录，密码使用加盐哈希保存。

## 部署结构

- `public/`：静态前端资源。
- `src/server.js`：静态文件服务、注册登录接口、日志数据接口。
- `data/`：运行期用户、会话和日志数据目录，不提交到仓库。
- `package.json`：启动和检查脚本。

后续如果用户量变大，可以把 `data/` JSON 存储替换为 SQLite、PostgreSQL 或 MySQL，前端接口不用大改。
